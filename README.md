# Project Readme

## Introduction

1. This project helps in running the code in a simulated environment of the `Donkey Simulator`. The model saved checkpoints and the relevant codes are provided. Please follow the below steps to download and run the code in the simulator.

## Installation & Setup

1. **Install simulator**

   Dowload the simulator from [Donkey Simulator](https://github.com/tawnkramer/gym-donkeycar/releases)

2. **Setup simulator**

   Open the simulator and select `generated_road`

3. **Install Python dependencies**

   ```bash
   pip install -r requirements.txt
   ```

4. **Running the code**
   ```bash
   python run_donkey.py
   ```

### After the following steps the car should be running in the simulator. There is also a visualisation provided for what the image that is being passed from the simulator to the model.
